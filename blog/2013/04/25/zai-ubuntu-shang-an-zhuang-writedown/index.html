    在 Ubuntu 上安裝 Writedown

    前些時間 @sofish 大神發佈了一個基於 node-webkit 的跨平臺 Markdown 編輯器 Writedown。
    但是這貨在 Ubuntu(或是Linux?) 下，會出現 localStorage 的無法修改的情況。剛好我最近也對 NanoDB 進行更新，支持了 Node.js 環境，可以進行文件存儲。
    可 sofish 只在 Github 上放出了 Mac OS X 下的安裝包，如果要在 Ubuntu 下安裝 Writedown， 你就需要自己進行安裝。

    1. 下載源碼

    你需要從 Github 上克隆(clone) Writedown 的源碼。

    然後從我的代碼庫中獲取更新。

    當然你也可以直接從我的 Fork Repo 中下載更新後的源碼。

    2. 安裝 node-webkit

    你需要安裝 node-webkit 以運行 Writedown。

    從 node-webkit 的 Github Repo 上獲得 Linux(x86/x64) 的壓縮文件，並建立鏈接。

    現在已經安裝好了 node-webkit。

    3. 安裝 Writedown

    我們回到剛纔下載 Writedown 的文件夾，並把它複製到我們的程序目錄下。

    現在就能在 Applications 中看到 Writedown 了。
